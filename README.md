# 🧠 AI-Powered Competitive Programming Platform  
A full-stack competitive programming platform built using the **MERN stack**, featuring a real-time code editor, AI-powered hints, Redis caching, and a secure authentication system.

> 🚀 Designed to simulate real-world coding platforms like LeetCode with AI-powered assistance.


# 🧰 Tech Stack Summary

- 💻 Frontend: React.js, Redux Toolkit, Tailwind CSS, Monaco Editor  
- ⚙️ Backend: Node.js, Express.js  
- 🗄️ Database: MongoDB  
- ⚡ Caching: Redis  
- 🤖 AI Integration: Google Gemini API  
- 🔐 Authentication: JWT + HTTP-only Cookies  
- ☁️ Media Storage: Cloudinary  



# ✨ Key Highlights

- 🔥 Built a scalable full-stack platform with real-world architecture  
- 🤖 Integrated AI-based hint generation reducing problem-solving time by ~30%  
- ⚡ Optimized performance using Redis caching (up to 70% faster responses)  
- 🔐 Implemented secure JWT-based authentication with role-based access  
- 📊 Developed admin dashboard for problem management  



# 🚀 Features

### 💻 Live Code Editor
- Monaco Editor integration  
- Real-time coding, testing, and submission  

### 🤖 AI Coding Assistant
- Uses Google Gemini API  
- Provides personalized hints based on user code  
- Avoids direct solutions to enhance learning  

### ⚡ Performance Optimization
- Redis caching for frequently accessed problems  
- Reduced database load and improved speed  

### 🔐 Authentication & Security
- JWT-based authentication  
- HTTP-only cookies (prevents XSS attacks)  
- Role-based access control  

### 🛠️ Admin Dashboard
- Add/edit coding problems  
- Upload video solutions via Cloudinary  



# 🧩 System Architecture

Frontend (React + Redux)  
   ↓  
Backend (Node.js + Express)  
   ↓  
Database (MongoDB)  
   ↓  
Caching Layer (Redis)  
   ↓  
External Services:  
   - Google Gemini API (AI hints)  
   - Cloudinary (media storage)
